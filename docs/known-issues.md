# Known Issues and Limitations

- No visual feedback is shown

## Chart Uses Demo Data
- Chart is rendered using a TradingView demo feed
- Data does not align with the token list or transaction table

## Transactions Tied to a Single Pair
- Swap table appears to show activity for one market only

## No Wallet Integration in Screener
- No connect wallet button
- No balance display

**Expected behavior**
