# Running the Sample App

## Prerequisites
- Node.js (LTS recommended)
- npm

## Install and Run

npm install
npm run dev

## Expected Behavior

- Frontend runs on http://localhost:3000
- Backend starts on port 5006
- App loads as a **read-only dashboard**
- Trade execution redirects externally to **ThetaSwap**

## Setup Notes

- Config directory warnings may appear during startup
- Webpack middleware deprecation warnings are non-blocking
